hexo搭建 2020-11-15

hexo

安装hexo （耗时有点久需要等待

将hexo安装到E/Workspaces/hexo

输入hexo g 和 hexo s（例如spark平台，git bash中CTRL+C = stop

$ hexo g # 生成
$ hexo s # 启动服务

在网页中输入localhost:4000（如图显示

接下来可以去官网选择主题https://hexo.io/themes/

（如果你觉的上图页面丑的话。。。。。。

我选择的是Melody

点击Melody 进入如下图

复制网址并回到git bash

在themes目录下有新的主题

返回hexo目录下的_config.yml，将其中的theme: landscape 改为 theme:melody

然后 hexo clean 加上hexo g 再hexo s 网址输入localhost:4000然后网页就没了/(ㄒoㄒ)/~~（如图

来到https://github.com/Molunerfinn/hexo-theme-melody看底下大佬发言。

懂了！ 先去看看版本号 hexo version

然后看到自己的是5.2.0版本 （”hexo”中文翻译是”十六进制”，所以建议看文档不要先翻译！）

npm安装hexo-renderer-pug hexo-renderer-stylus

install melody（讲道理，这步在上一步之前

最后 hexo clean hexo g hexo s 实现效果 （2020/11/15 3：03）
